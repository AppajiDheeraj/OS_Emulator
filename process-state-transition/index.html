SIMULATION LOGIC

Round Robin (RR) preempts the currently running process if its burst exceeds the Quantum Limit, seamlessly dispatching the next process in the Ready Queue.

Processes occasionally trigger I/O Wait Requests, shifting them into a blocked state until hardware interrupts acknowledge I/O completion.
