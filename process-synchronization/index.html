SIMULATION LOGIC

Producers need semEmpty > 0 to create an item.
Consumers need semFull > 0 to consume an item.

Both must acquire the Mutex lock before modifying the buffer to prevent race conditions. If the Mutex is securely held, subsequent processes are pushed into the Wait Queue (FIFO).
